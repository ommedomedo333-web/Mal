## Color Palette (60-30-10 Rule)

### Primary Colors
- **60% - Deep Forest Green** `#003e31`
  - Main background color
  - Creates a premium, natural feel
  - Used for: App background, dark surfaces

- **30% - Warm Cream** `#fee9d1`
  - Secondary color for cards and text
  - Provides excellent contrast
  - Used for: Product cards, text, surfaces, buttons

- **10% - Rust Orange** `#db6a28`
  - Accent color for CTAs and highlights
  - Draws attention to important actions
  - Used for: Active states, badges, primary buttons

### Supporting Colors
- **Surface Dark**: `#004d3e` (Slightly lighter than bg)
- **Text Primary**: `#fee9d1` (Cream)
- **Text Secondary**: `rgba(254, 233, 209, 0.6)` (Cream 60%)

## Design Principles

### 1. Visual Hierarchy
- Background (60%): Deep green creates immersive environment
- Content (30%): Cream cards pop against dark background
- Actions (10%): Orange accents guide user attention

### 2. Typography
- **Arabic**: Cairo, Tajawal (RTL support)
- **English**: Poppins, DM Sans
- **Hierarchy**: Bold weights for headings, regular for body

### 3. Components

#### Bottom Navigation
- Floating design with glassmorphism
- Orange indicator for active state
- Cream text labels
- Smooth animations

#### Product Cards
- White/Cream background
- Orange accent for prices and badges
- Deep green for category backgrounds
- Rounded corners (24px)

#### Buttons
- Primary: Orange background, white text
- Secondary: Cream background, dark text
- Ghost: Transparent with cream border

### 4. Spacing System
- Base unit: 4px
- Small: 8px
- Medium: 16px
- Large: 24px
- XL: 32px

### 5. Border Radius
- Small: 8px
- Medium: 16px
- Large: 24px
- XL: 32px
- Full: 9999px (pills)
