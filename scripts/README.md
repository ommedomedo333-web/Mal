# دليل استيراد البيانات إلى Supabase

## الخطوات:

### 1. تحضير البيانات
قم بنسخ البيانات JSON الكاملة التي أرسلتها وحفظها في ملف:
```
scripts/full-products-data.json
```

### 2. تشغيل السكريبت
```bash
node scripts/import-to-supabase.js
```

### 3. التحقق من النتائج
- سيتم إضافة 12 قسم
- سيتم إضافة 168 منتج
- كل منتج سيحتوي على:
  - الاسم العربي
  - السعر
  - الوحدة
  - الباركود
  - الصورة
  - الوصف
  - التقييم

## ملاحظات مهمة:

1. **الألوان**: جميع الأقسام تستخدم نظام الألوان الجديد:
   - اللون الأساسي: `#003e31` (60%)
   - اللون الثانوي: `#fee9d1` (30%)
   - لون التمييز: `#db6a28` (10%)

2. **الصور**: جميع المنتجات تحتوي على صور من Unsplash

3. **النقاط**: يتم حساب النقاط تلقائياً (السعر × 10)

4. **المخزون**: جميع المنتجات متوفرة بكمية 100 قطعة

## البيانات المطلوبة في .env:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
```
